# Beauty Corner

A professional, modern website for Beauty Corner - a professional hair care and styling salon.

## Description

Beauty Corner is a responsive web application designed to showcase a professional hair salon's services, portfolio, and contact information. The website provides potential clients with an overview of services offered and an easy way to get in touch.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern Navigation**: Sticky header with smooth navigation to different sections
- **Service Showcase**: Display of four main services (Hair Cutting, Hair Coloring, Hair Styling, Hair Treatment)
- **About Section**: Information about the salon and its commitment to customers
- **Gallery**: Grid layout for showcasing salon work and portfolio images
- **Contact Information**: Easy access to phone, email, and physical address
- **Call-to-Action Button**: "Book an Appointment" button for customer engagement
- **Professional Styling**: Elegant color scheme with gold and dark gray accents

## Technologies Used

- **HTML5**: Semantic markup for page structure
- **CSS3**: Modern styling with CSS variables, flexbox, and gradients
- **Responsive Design**: Mobile-first approach with viewport meta tag

## Sections

1. **Header & Navigation**: Sticky navigation bar with links to all main sections
2. **Hero Section**: Welcome message and call-to-action button
3. **Services**: Grid layout showcasing four main hair services
4. **About Us**: Information about the salon's mission and values
5. **Gallery**: Grid of portfolio images (currently using placeholders)
6. **Contact**: Contact details including phone, email, and address
7. **Footer**: Additional footer content

## Getting Started

1. Open `index.html` in a web browser to view the website
2. Navigate using the menu to explore different sections
3. Update placeholder images in the gallery with actual salon photos
4. Customize contact information in the Contact section as needed
